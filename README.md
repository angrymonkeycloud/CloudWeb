# Cloud Web

## Url
 
Base Url: <https://cdn.amcapi.com/cloudweb/>

Complete Url: {BaseUrl}/{Version}/Filename.

Example: <https://cdn.amcapi.com/cloudweb/1.0.2/cloud-base.less>

## Files

- cloud-base.less
- cloud-variables.less
- cloud-sections.less
- cloud-header.less

## Import Less

Import .less URL as follows:

```less

@import url('...');

```
